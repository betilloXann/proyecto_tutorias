# Stack Tecnológico

Este prototipo se está construyendo bajo el paradigma *multiplataforma* con el siguiente *stack* tecnológico clave:

| Componente | Tecnología Principal | Propósito |
| :--- | :--- | :--- |
| **Frontend (Móvil & Web)** | **Dart / Flutter** | Desarrollo de interfaces nativas para Android/iOS y la interfaz web administrativa. |
| **Backend / Servidor** | **Firebase** | Manejo de la lógica de negocio, autenticación de usuarios y APIs. |
| **Base de Datos** | **Firestore** | Almacenamiento seguro de datos de alumnos, tutores y registros de sesiones. |
| **Control de Versiones** | **Git / GitHub** | Colaboración y gestión del código fuente. |
